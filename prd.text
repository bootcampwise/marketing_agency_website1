# **1. Product Overview**

This website is a marketing agency platform built in **Next.js 14 (App Router)** with content fully managed in **Prismic CMS** using **Slice Machine**.
The design follows your Figma layout with sections for:

* Home
* Services
* About Us
* Blogs
* Blog Details
* Pricing
* Case Studies
* Detailed Service Page (Digital Marketing)
* Contact (footer form)

The goal is a **modern, responsive, CMS-editable website** optimized for performance and SEO.

---

# **2. Tech Stack**

### **Core**

* **Next.js 14 App Router**
* **Prismic CMS** (Repository + Custom Types + Slices)
* **Slice Machine** for component-driven development
* **TypeScript**
* **Tailwind CSS** (recommended for speed)

### **Hosting**

* **Vercel** (recommended)

### **Integrations**

* Prismic API (GraphQuery)
* Contact form handler (serverless API route)
* Vercel Image Optimization

# **3. Routing Structure (Next.js App Router)**

```
/app
  /page.tsx                   ‚Üí Home
  /services/page.tsx          ‚Üí Services listing
  /services/[uid]/page.tsx    ‚Üí Single service
  /about/page.tsx             ‚Üí About
  /blogs/page.tsx             ‚Üí Blog listing
  /blogs/[uid]/page.tsx       ‚Üí Blog single
  /pricing/page.tsx           ‚Üí Pricing
  /case-studies/page.tsx      ‚Üí Case Study listing
  /case-studies/[uid]/page.tsx‚Üí Case Study single
  /api/contact/route.ts       ‚Üí Form submission
```

All content dynamically served from **Prismic**.

# **6. Functional Requirements**

### **6.1 CMS Integration**

* Fully dynamic content
* All texts editable through Prismic
* No hardcoded content except layout

### **6.2 Image Handling**

* Use Prismic Image field
* Render via Next.js `<Image />`

### **6.3 Contact Form**

* POST ‚Üí Next.js route handler
* SPAM protection via hidden honey-pot or hCaptcha

### **6.4 SEO**

* Dynamic metadata per Prismic page
* `generateMetadata()` in each route
* OpenGraph images
* Sitemap from `next-sitemap`

---

# **7. Performance Requirements**

* Pre-render via **Static Generation** where possible
* Revalidate using Prismic webhooks ‚Üí ISR
* Target Lighthouse scores:

  * Performance: 90+
  * Accessibility: 95+
  * SEO: 100
