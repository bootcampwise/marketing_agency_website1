Build a modern and responsive marketing agency website showcasing
services, pricing, use cases, and blogs. The site should communicate
the brand identity clearly, attract clients, and ensure a seamless,
engaging experience across all devices.

Task Checklist

Self QA (Before Ticket Move to Review)\*
• Test all pages, links, and navigation.
• Remove console logs, unused imports, and assets.
• Verify all images and sections load correctly on devices.
• Attach deployed link in ticket.
• Create LinkedIn post with hashtags #TechloSet #BootcampWise.

Todos\*
• Home: Hero, key services, use cases, workflow.
• Services: Grid layout of offered services.
• Service Detail: Dynamic info page.
• About: Company background & team.
• Blogs: List with categories.
• Blog Detail: Full blog page.
• Pricing: Packages.
• Use Cases: Real success stories.

Requirements\*
• Fully responsive design.
• Use reusable components for header, footer, and cards.
• Ensure smooth navigation.
• Maintain consistent typography & spacing.
• Optimize for performance and mobile devices.

Coding Standards\*
• PascalCase for components, camelCase for variables.
• Modular and reusable structure.
• Clean, readable code with comments.
• Avoid unused imports and redundant assets.
• Follow responsive and accessibility best practices.
